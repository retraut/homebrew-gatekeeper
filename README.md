# Homebrew Tap for Gatekeeper

This is a Homebrew tap for [Gatekeeper](https://github.com/retraut/gatekeeper), a service authentication status monitor with daemon, CLI, tmux integration, and macOS GUI.

## Installation

```bash
brew tap retraut/gatekeeper
brew install gatekeeper
```

## Usage

```bash
gatekeeper --help
```
